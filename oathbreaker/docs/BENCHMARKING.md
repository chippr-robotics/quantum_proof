# The Oathbreaker Scale — Quantum Benchmarking Specification

## Overview

The Oathbreaker circuit serves as a standardized benchmark for quantum hardware. Score your quantum computer by which **Oath curve** it can crack. As quantum computers scale, this provides a ready-to-execute program at known difficulty.

## The Oathbreaker Scale

In the table below, **Curve** names the toy Goldilocks curve family used for that benchmark tier, **Field Size** is the approximate bit length of the underlying prime field, and **Security** is the approximate classical discrete-log security level for that tier.

All resource counts below are **measured** from actual circuit construction (Oath-8/16/32) or **projected** via the Karatsuba O(n^2.585) scaling model (Oath-64+). The circuit uses Jacobian projective coordinates, Karatsuba multiplication, symmetry-optimized squaring, Binary GCD inversion, and windowed scalar multiplication with one-hot QROM decode.

| Level | Curve | Field Size | Security | Qubits (Bennett) | Qubits (meas-based) | Toffoli | Classical Solve Time | Quantum Target |
|-------|-------|------------|----------|-------------------|---------------------|---------|----------------------|----------------|
| **Oath-8** | 8-bit Goldilocks | 8-bit | ~4-bit | 210 | 186 | 112K | Instant | Proof of concept |
| **Oath-16** | 16-bit Goldilocks | 16-bit | ~8-bit | 402 | 370 | 929K | Instant | Near-term devices |
| **Oath-32** | 32-bit Goldilocks | 32-bit | ~16-bit | 1,026 | 738 | 5.76M | Milliseconds | Medium-term milestone |
| **Oath-64** | 64-bit Goldilocks | 64-bit | ~32-bit | ~2,052 | ~1,474 | ~35M | Minutes (Pollard rho) | Full benchmark target |

Oath-8/16/32 are measured from actual circuit construction with proper ancilla reuse. Oath-64 is projected (circuit construction exceeds CI memory at ~3 GB). "Bennett" qubits are the peak with standard reversible uncomputation; "meas-based" estimates the qubit count achievable with mid-circuit measurement and classical feedforward.

### Cost Attribution (Oath-32, w=8)

| Subsystem | Toffoli | Share |
|-----------|---------|-------|
| Doublings | 4,541,952 | 80.2% |
| Mixed additions | 971,616 | 17.1% |
| Inversion (BGCD) | 107,008 | 1.9% |
| Affine recovery | 36,868 | 0.7% |
| QROM decode/load | 8,160 | 0.1% |

Each Oath curve is an elliptic curve over a Goldilocks-form prime field, with verified prime order and safe parameters generated by SageMath. All parameters are independently verified via SageMath's SEA algorithm in CI.

## Window Size Optimization

The benchmark includes a window-size sweep for each tier. Results confirm w=8 is monotonically best for Toffoli across all tiers:

| Tier | w=1 Toffoli | w=2 Toffoli | w=4 Toffoli | w=8 Toffoli |
|------|------------|------------|------------|------------|
| Oath-32 | 13.4M | 9.5M | 7.6M | 5.76M |

The tradeoff is purely in qubits — w=8 adds ~20-50% more qubits from the 2^w one-hot selection register.

## Input Format

Each benchmark instance consists of:
1. Oath curve parameters (a, b, p, generator G)
2. Target point Q = [k]G
3. The QASM circuit description for that difficulty level

## Expected Output

The discrete logarithm k such that Q = [k]G — the "oath" that was broken.

## Scoring

Oathbreaker Score = f(runtime, error_rate, qubit_count):

- **Runtime**: Wall-clock time to produce the correct discrete logarithm
- **Error rate**: Fraction of runs producing incorrect output
- **Qubit count**: Physical qubits used (including error correction overhead)

A machine's **Oathbreaker rating** is the highest Oath-N level it can solve:
- If you can crack Oath-8, your machine rates **OB-8**
- If you can crack Oath-16, your machine rates **OB-16**
- And so on up to **OB-64** for the full benchmark

## Verification

All Oath levels can be verified classically:
- Compute [k]G and confirm it equals Q
- For Oath-32 and above: independent verification via Pollard's rho
- For all levels: verified against the SP1 Groth16 proof
- Curve parameters independently verified via SageMath SEA in CI

## QASM Export

```bash
# Export a single tier (default: Oath-16)
cargo run --release -p benchmark -- export-qasm

# Export all tiers
cargo run --release -p benchmark -- export-all-qasm
```

Produces OpenQASM 3.0 files for each Oath level, loadable in Qiskit, Cirq, and other frameworks. Each export includes a companion `.stats.json` with resource counts.
